# 💍 Tilak Jewellers – Premium eCommerce Website

> **"Celebrate Every Moment with Timeless Jewellery"** — Since 1996 ✨

A luxury jewellery eCommerce website for **Tilak Jewellers**, Main Kasna Road, Greater Noida.

## 🚀 Tech Stack

- **Next.js 14** (App Router) + **TypeScript**
- **Tailwind CSS** + **Framer Motion**
- **next-themes** for Dark Mode
- **Lucide React** + **React Icons**

## ✨ Features

- 🌟 Floating golden particle hero animation
- 📱 Fully responsive (mobile, tablet, desktop)
- 🌙 Dark Mode toggle
- 💬 WhatsApp integration (floating + per-product)
- 📸 Instagram gallery — [@tilak_jewellers1](https://www.instagram.com/tilak_jewellers1)
- 🎨 Custom jewellery design form → WhatsApp
- ⭐ 5-star testimonials slider
- ❓ FAQ accordion
- 📍 Google Maps + contact form
- 📧 Newsletter subscription

## 🏃 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
